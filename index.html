My Shortcut Keys

Windows

  • Win + V → Clipboard history
  • Win + Shift + S → Screenshot
  • Win + Ctrl + D → New desktop
  • Win + Ctrl + Left/Right → Switch desktops
  • Win + . → Emoji panel

Chrome

  • Ctrl + Shift + T → Reopen tab
  • Ctrl + L → Address bar
  • Ctrl + Tab → Switch tab
  • Ctrl + Shift + N → Incognito
  • Ctrl + D → Bookmark
